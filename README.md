# Crawlbuddy

An experiment into crawling and parsing as described at http://goo.gl/3MW3x

## Installation

Add this line to your application's Gemfile:

    gem 'crawlbuddy'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install crawlbuddy
